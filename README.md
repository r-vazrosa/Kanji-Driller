# KanjiDriller

KanjiDriller is a desktop application designed to help learners practice **kanji meanings and readings** using **JLPT** or **WaniKani**â€“style level systems.

## Features
- Filter kanji by level and list size
- Support for JLPT and WaniKani level systems (meanings and readings)
- Randomized subsets from the filtered kanji list
- Session-based tracking of incorrect answers
- Customizable user profile (username and profile image)

## Data Source
Kanji data is provided by the excellent dataset maintained by  
[davidluzgouveia](https://github.com/davidluzgouveia/kanji-data).

## Status
This project is in its **initial release** and under active development.
